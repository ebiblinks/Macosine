Installation

Extract the Main Zip File , You'll see these folders and files:

  • doc-file - Help file
  • main-file - Main Template Folder
    • assets - CSS | Images | JS Folder with all files
    • 17 HTML Files - All Pages Files

We Provide 17 HTML File :

  1. index.html (Home Page)
  2. index2.html
  3. about-us.html
  4. services.html

  13. blog-grid.html
  14. blog-list.html

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the Induris - Factory & Industrial HTML Template > main-file folder to find all the files. You will need to upload these files to your hosting web server using FTP in order to use it on your own website.
  2. You need to upload all files inside the website root.
    • Assets Folder
    • 17 HTML Files
  3. You are now ready to start adding your Content & Images and generate your beautiful brand new website for your awesome users.

CSS Structure

If you would like to edit a specific section of the site, then go to css folder and open style.css file, and then scroll down until you find the appropriate style that needs to be edited. These are the css files that are loaded into templates in top of document.


Open the css folder and Find all CSS files.

  • bootstrap.min.css - Bootstrap Style
  • swiper-bundle.min.css - Swiper Slider Style
  • magnific-popup.min.css - Magnific Popup Style
  • animate.min.css - Animation Style
  • style.css - Theme Common Styles With Responsive Style In It
